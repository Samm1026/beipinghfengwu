# 北平风物记 - 北京文化漫游路线网站

## 项目简介
北平风物记是一个关于北京文化漫游路线的网站，旨在为用户提供有叙事深度的文化漫游体验。网站围绕用户的深度文化漫游需求，搭建了四大核心功能模块，打造沉浸式体验闭环。

## 核心功能

### 1. 文化内容呈现
- 北京文化漫游地图
- 精品景点设施展示
- 历史故事深度解析

### 2. 氛围入口
- 页面加载后自动播放15秒背景白噪音
- 简洁文字引导，营造漫步氛围
- 支持手动关闭音频

### 3. 彩蛋式热点交互
- 路线图片中嵌入透明可点击的交互点
- 点击后以浮层形式展示历史老照片或相关短诗
- 增加文化探索的趣味性

### 4. 社交分享功能
- 页面底部设置分享按钮
- 支持一键生成社交分享图
- 便于用户分享心仪的路线

## 页面说明

### 首页
- 开场氛围：页面加载时的引导动画和背景音频
- 快速入口：直达各主要功能模块
- 精品推荐：精选的文化漫游路线展示

### 漫游地图
- 地图展示：简化的北京文化地图
- 地图标记：各主要景点的标记点
- 路线预览：推荐路线的简要介绍

### 路线策展
- 分类浏览：按主题分类的路线列表
- 路线详情：每条路线的详细信息
- 专题策划：深度文化内容展示

## 使用方法

1. 克隆或下载项目到本地
2. 在浏览器中打开 `index.html` 文件
3. 点击"开始探索"按钮进入主页面
4. 浏览不同的功能模块，体验各种交互效果
